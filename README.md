🚀 Yapay Zeka ve Teknoloji Akademisi - Proje Portfolyosu

Bu depo, Yapay Zeka ve Teknoloji Akademisi bünyesinde tamamladığım Veri Bilimi ve Makine Öğrenmesi projelerini içermektedir. Projelerde veri temizleme, görselleştirme ve tahminleme modelleri üzerine odaklanılmıştır.

📑 Proje Listesi
1. 📊 İK Analitiği: Çalışan Elde Tutma (Mezuniyet Projesi)
Dosya: Mezuniyet Proje (1).ipynb

Özet: Salifort Motors şirketindeki çalışanların işten ayrılma durumlarını tahmin etmek için geliştirilmiştir.

Yöntem: Lojistik Regresyon ve Random Forest modelleri kıyaslanmıştır.

English: Predicted employee attrition using classification models to improve retention strategies.


2. 🛒 Datathon: Ürün Fiyat Tahmini
Dosya: DatathonProje (2).ipynb

Özet: Farklı market ve kategorilerdeki ürünlerin fiyat değişimlerini tahmin eden bir regresyon projesidir.

Yöntem: LightGBM algoritması kullanılarak yüksek doğruluklu fiyat tahminleri yapılmıştır.

English: Regression-based price forecasting for grocery products using LightGBM.



3. 🚕 Automatidata: Taxi Bahşiş Tahmini
Dosya: Automatidata_Projesi (1).ipynb

Özet: NYC Taxi verilerini kullanarak, yolcuların cömert bahşiş (≥%20) bırakıp bırakmayacağını tahmin eder.

Yöntem: XGBoost ve Random Forest algoritmaları ile özellik önemi (Feature Importance) analizi yapılmıştır.

English: Binary classification model to predict generous tipping behavior in NYC taxis.



🛠️ Yetkinlikler & Araçlar
Diller: Python

Kütüphaneler: Pandas, NumPy, Scikit-learn, XGBoost, LightGBM, Seaborn, Matplotlib.

Süreçler: Keşifsel Veri Analizi (EDA), Özellik Mühendisliği (Feature Engineering), Hiperparametre Optimizasyonu.
